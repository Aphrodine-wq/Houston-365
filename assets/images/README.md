# Placeholder Images

This directory contains branded placeholder images for posts without featured images.

## Required Placeholders

Create the following placeholder images with Houston Texans branding (Battle Red #A71930, Deep Steel Blue #03202F):

### 1. `hero-placeholder.jpg`
- **Dimensions**: 1600×900 (16:9)
- **Usage**: Hero slider when post has no featured image
- **Design**: Houston Texans logo centered on deep steel blue background, "Featured Content" text

### 2. `card-placeholder.jpg`
- **Dimensions**: 480×320 (3:2)
- **Usage**: Post cards on homepage and archives
- **Design**: Houston Texans wordmark on light gray background, "News" text

### 3. `podcast-placeholder.jpg`
- **Dimensions**: 150×150 (1:1 square)
- **Usage**: Podcast module in sidebar
- **Design**: Microphone icon with Houston Texans colors, "Podcast" text

## Design Guidelines

- Use solid backgrounds or subtle patterns
- Include Houston Texans branding (logo, colors, wordmark)
- Ensure text is readable (4.5:1 contrast ratio minimum)
- Keep file sizes optimized (<100KB each)
- Save as progressive JPEGs for better web performance

## Quick Generation (Placeholder Tools)

If you need temporary placeholders immediately:
1. Use https://placehold.co/1600x900/03202F/FFFFFF/jpeg?text=Houston365
2. Replace with branded versions before launch

## Files

- `hero-placeholder.jpg` - 1600×900 hero image

## Implementation

These placeholders are used automatically by the theme when a post lacks a featured image. No manual intervention required - they provide graceful fallback for missing images.
